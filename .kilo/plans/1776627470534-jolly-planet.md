# Plan: Dynamic Main Menu Navigation Per Controller

## Problem
The navigation sidebar in `index.html` and functions in `app.js` are hardcoded with the same 5 menu options, but each controller can have different menu options in different order.

## Current State
- **Data** (`controllers.js`): Each controller already defines its own `mainMenu` with different options
- **UI** (`index.html`): Hardcoded 5 nav buttons (lines 23-53)
- **JS** (`app.js`): Hardcoded arrays in `setNavActive` (line 84) and `updateNavLabels` (lines 50-56)

## Solution

### 1. Make Navigation Dynamic in index.html
Replace hardcoded nav buttons with a container that gets populated dynamically:

```html
<!-- Replace lines 21-54 in index.html -->
<div class="nav-section">
  <div class="nav-label">Main menu</div>
  <div id="main-nav-container"></div>
</div>
```

### 2. Add Navigation Rendering Function in app.js
Add a function to render nav items based on the current controller's `mainMenu`:

```javascript
function renderMainNav() {
  const container = document.getElementById('main-nav-container');
  const mainMenu = CONTROLLERS[currentControllerId]?.mainMenu;
  if (!container || !mainMenu) return;
  
  let html = '';
  Object.keys(mainMenu).forEach(key => {
    const item = mainMenu[key];
    html += `<button class="nav-item" id="nav-${key}" onclick="showSection('${key}')">
      <span class="nav-label-text">${item.label}</span>
    </button>`;
  });
  container.innerHTML = html;
}
```

### 3. Update app.js Functions to Use Dynamic Keys

**updateNavLabels** (lines 46-72): Remove hardcoded navMap, use controller's mainMenu keys directly:

```javascript
function updateNavLabels() {
  const mainMenu = CONTROLLERS[currentControllerId]?.mainMenu;
  if (!mainMenu) return;
  
  Object.keys(mainMenu).forEach(key => {
    const btn = document.getElementById('nav-' + key);
    if (btn) {
      const label = mainMenu[key].label;
      const labelSpan = btn.querySelector('.nav-label-text');
      if (labelSpan) labelSpan.textContent = label;
    }
  });
}
```

**setNavActive** (lines 83-90): Use dynamic keys from the controller:

```javascript
function setNavActive(sectionId) {
  const mainMenu = CONTROLLERS[currentControllerId]?.mainMenu;
  if (!mainMenu) return;
  
  Object.keys(mainMenu).forEach(key => {
    const el = document.getElementById('nav-' + key);
    if (el) el.classList.remove('active');
  });
  const el = document.getElementById('nav-' + sectionId);
  if (el) el.classList.add('active');
}
```

### 4. Call renderMainNav on Controller Switch
In `initController()` (after line 14) and `switchController()` (after line 79), add:

```javascript
renderMainNav();
```
